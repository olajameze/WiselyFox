# Pedagogy

## Mastery engine

- Spaced repetition via `nextReviewDate()`
- Difficulty adjusts with `adjustDifficulty()` based on mastery score

## Assessment

- Entrance assessment covers reading, maths, logic, memory, attention
- Levels: Foundation, Emerging, Developing, Advanced

## Study skills courses

Seeded subjects include focus techniques, note-taking scaffolds, and emotional regulation for learning.
