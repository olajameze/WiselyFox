# Inclusive Learning Research

WiselyFox accommodations trace to established frameworks:

## Universal Design for Learning (UDL), CAST

- Multiple means of representation, action, and engagement in lesson templates

## Rosenshine's Principles

- Small steps, guided practice, frequent checks

## Condition-informed design (non-diagnostic)

| Profile | UI adaptations |
|---------|----------------|
| ADHD | Short sessions, breaks, immediate feedback, minimal distractions |
| Autism | Visual schedules, predictable layout, explicit steps |
| Dyslexia | Spacing, optional dyslexia friendly text, TTS-ready content |
| Anxiety | Low-pressure mode, gentle errors, optional timer off |

## Guardrails

- WiselyFox is an education tool, not a medical device
- Strengths based language only
- Parent controls all sensitive profile data

See also `docs/PEDAGOGY.md`.
